# Examples

Ready-to-run chaos experiments. Each example includes the YAML manifest and a
step-by-step README.

| Example | Scenario | Description |
|---------|----------|-------------|
| [tm-pod-kill](tm-pod-kill/) | `TaskManagerPodKill` | Kill one or more TaskManager pods and observe Flink recovery |

---

## How to use

1. Install the operator — see [Quick Start](../docs/guidelines/quick-start.md)
2. Browse to the example folder
3. Edit the YAML to match your namespace and labels
4. Follow the README in that folder
